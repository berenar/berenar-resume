# Personal Resume Web App

This is my online resume webapp built using ReactJS, NextJS and other awesome technologies.
## Installation

You can clone this project and use it to build your own resume.

```bash
git clone
cd berenar-resume
```

## Usage

```bash
npm run dev
```
